# 因幡神社

因幡神社へようこそ!

## デプロイ

- [**Node.js**](https://nodejs.org/ja/) v12.13+
- [**Yarn**](https://yarnpkg.com/ja/)

```console
$ git clone https://github.com/inabagumi/inaba-jinja.git
$ cd inaba-jinja
$ yarn install --frozen-lockfile
$ yarn build
$ yarn start
```
